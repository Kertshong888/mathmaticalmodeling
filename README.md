# mathmaticalmodeling
# Delivery Fleet Cost Optimisation

A Python-based mathematical modelling project that analyses delivery company 
operational costs and identifies optimal staffing levels.

## Tools Used
Python, Pandas, NumPy, Matplotlib

## What it does
- Builds a regression cost model from 500 days of operational data
- Uses gradient descent to find cost-minimising van and employee numbers
- Applies bootstrap resampling to quantify prediction uncertainty

## Key Result
Identified optimal staffing of 9 vans and 9 employees, reducing predicted 
daily cost from £2,240 to £1,837
